Description

Gespeaker is a free GTK+ frontend for espeak. It allows you to play a text in many languages with settings for voice, pitch, volume and speed. The text readed can also be recorded to WAV file for future listening.

The text can be read with many available languages.

To hear some samples of recorded voices please see the demo voices page.

Installation

Last stable version is 0.8.2 with the MBROLA voices support for a new speech experience. Version changes are available on the Changelog page.

New version 0.8.1 has plugins support to add external activities as well interaction with aMSN, Kopete, Emesene, Pidgin, Empathy messenger. You can get them from the Download page.

Enable enhanced voice support

Since version 0.6 it supports the speech synthesizer MBROLA for a better speech experience.

See the MBROLA page for instructions on enable the enhanced MBROLA voice support.

Translations

Gespeaker is actually translated in the following languages:

  • French
