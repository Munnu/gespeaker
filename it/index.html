Descrizione

Gespeaker è un'interfaccia GTK+ a espeak. Permette di ascoltare un testo in varie lingue con impostazioni per voce, intonazione, volume e velocità. Il testo letto può anche essere registrato in file WAV per ascoltarlo successivamente.

Il testo può essere letto con tante lingue disponibili.

Per ascoltare alcuni esempio di voci registrate vedere la pagina voci dimostrative.

Installazione

I rilasci delle versioni stabili con codice sorgente e pacchetti precompilati sono presenti nella pagina Scarica. Per informazioni sull'installazione da pacchetti o con codice sorgente fare riferimento alla pagina Installazione.

Il codice sorgente di Gespeaker è disponibile su Github con la versione attualmente in sviluppo.

Attivare il supporto vocale avanzato

Dalla versione 0.6 Gespeaker supporta il sintetizzatore vocale MBROLA per una migliore esperienza vocale.

Consultare la pagina MBROLA per istruzioni su come attivare il supporto vocale avanzato MBROLA.

Traduzioni

Gespeaker è attualmente tradotto nelle seguenti lingue:

  • Inglese
  • Francese
  • Italiano
  • Spagnolo
  • Polacco

Traduttori sono necessari, per favore aiutami a tradurre Gespeaker nella tua lingua.

Prego inviate le vostre traduzioni a bugtracker di Github oppure invia una richiesta pull su Github.
